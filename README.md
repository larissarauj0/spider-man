# 🕷️ Spider-Man: Miles Morales Landing Page

Uma landing page inspirada no universo de **Spider-Man: Miles Morales**, desenvolvida com **Next.js**, **TypeScript**, **Tailwind CSS** e animações utilizando **Framer Motion**.

O projeto foi criado com foco em prática de **Frontend moderno**, componentização, responsividade e experiência visual.


## 🚀 Tecnologias utilizadas

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion

---

## 🎯 Objetivos do projeto

Este projeto foi desenvolvido para praticar:

- Estruturação de projetos com Next.js
- Componentização em React
- Responsividade com Tailwind CSS
- Animações de entrada com Framer Motion
- Organização de layout e camadas (`z-index`)
- Construção de interfaces inspiradas em designs cinematográficos

---



## ⚙️ Como rodar o projeto localmente

### 1. Clone o repositório

```bash
git clone URL_DO_REPOSITORIO
```

### 2. Entre na pasta

```bash
cd spider-man
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Rode o projeto

```bash
npm run dev
```

Abra no navegador:

```txt
http://localhost:3000
```
